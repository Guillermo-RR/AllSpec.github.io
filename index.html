AllSpec

Tratamiento unificado de datos espectroscópicos y electroquímicos

¿Qué es AllSpec?

AllSpec es una herramienta libre para el tratamiento y análisis de datos obtenidos mediante técnicas como UV-Vis, fluorescencia, espectroscopía IR, difracción de rayos X en polvos y voltametría (CV, LV, cronoamperometría, disco rotatorio). Desarrollado con un enfoque educativo, permite ajustar, visualizar y exportar resultados de manera intuitiva.

Características principales

  • Lectura de archivos en distintos formatos
  • Soporte multitécnica en una misma plataforma
  • Exportación de figuras y base de datos
  • Interfaz gráfica amigable
